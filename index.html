Introduction to Git


What is Version Control

Version Control, also known as Revision Control or Source Code Management, is the management of changes to documents, computer programs, websites or other collection of information. These changes are often identified as numbers or letter code, termed as revision number. Each revision is associated with meta data like timestamp, author(a person who made changes), etc. Revisions can be compared, restored or merged.
